# .gitignore.txt
# This file has been created to get around an open issue with NPM
# where NPM will not publish a real .gitignore file. This causes the
# published @ey/voltron package to be missing the .gitignore, which keeps
# @ey/generator-voltron from properly creating a new project with an
# intact .gitignore, since there's nothing for it to copy.
# The solution is to create this file, then have the generator manually
# change its name once the file has been copied over.
